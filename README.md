# 🎯 Smart Recruitment Assistant

## 📌 Project Overview

Smart Recruitment Assistant is an AI-powered recruitment support system that helps organizations streamline the hiring process by automatically evaluating, scoring, and ranking job applicants.

The system analyzes multiple candidate attributes such as:

* CGPA
* Aptitude Score
* Technical Skills
* Communication Skills
* Certifications
* Internships
* Projects
* Work Experience
* Interview Performance

Using Machine Learning algorithms, the application predicts whether a candidate is likely to be selected or rejected and generates a Hiring Score to support data-driven recruitment decisions.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Recruitment-Assistant.git
cd Smart-Recruitment-Assistant
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Generate Dataset

```bash
py generate_dataset.py
```

### Train Model

```bash
py train_model.py
```

### Run Application

```bash
py -m streamlit run app.py
```
